input tag

<input type="입력타입" name="input 요소의 이름" value="입력 요소의 초기값">
type: text, password, tel, number, url, search, email, checkbox, radio, file, button, image(버튼요소 생성. src 속성 사용, alt x), hidden, date, datetime-local, month, week, time, range(범위), color, submit(폼 전송 역할 버튼), reset(입력값 초기화 버튼)

label 태그: 폼 태그 안에서 사용하는 상호작용 요소에 이름을 붙일 때 사용 (암묵적/명시적)
